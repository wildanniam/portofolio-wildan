## Capture rules for final production assets

- Recapture after a case-study flow is verified and before launch.
- Use a reproducible viewport and known data state.
- Record whether the media shows seeded, test, testnet, or production data.
- Redact wallets, emails, private identifiers, API secrets, and irrelevant browser context.
- Never use a screenshot's visible metric as a portfolio claim without a separate source record.
- Keep an original lossless capture and export intentional desktop/mobile derivatives.
